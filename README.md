## Installing project dependencies
#### Requirements:
- Make sure [`poetry`](https://python-poetry.org/docs/#installation) is installed.
- [`Update poetry`](https://python-poetry.org/docs/#updating-poetry)
  ```bash
  poetry self update
  ```

#### Installation
```bash
make install
```

## Start apps:
- `./server`
- `./mlmodels`
- `./mlpipeline`
